#  Digital Processing of Biosignals

Repository of scripts and materials used in the **Digital Processing of Biosignals** and **Artificial Inteligence for Biomedical Engineering Applications** courses. Contains practical implementations of acquisition, analysis, and processing techniques for biomedical signals such as ECG, EEG, and EMG, as well as basic concepts for Artificial Intelligence, including Bioinformatics, Fuzzy logic, and Deep Learning (ANN, CNN).

---

##  Contents

- Signal acquisition and visualization scripts
- Digital filter implementations (low-pass, high-pass, band-pass, notch)
- Frequency domain analysis (FFT, STFT, Wavelet)
- ECG, EEG, and/or EMG signal processing
- Bioinformatics
- Fuzzy logic
- Deep learning
- Lab exercises and assignments

---

##  Technologies & Libraries

- **Python**
- `numpy` — numerical operations and transforms
- `scipy` — filter design and signal processing
- `matplotlib` — signal visualization
- `pandas` — tabular data handling
- `tensorflow` — artificial intelligence and deep learning methods


---

##  Getting Started

1. Clone the repository:

```bash
git clone https://github.com/jedesvaz/Clase-de-Procesamiento-Digital-de-Bioseales.git
cd Clase-de-Procesamiento-Digital-de-Bioseales
```

2. Install dependencies:

```bash
pip install numpy scipy matplotlib pandas
```

> A virtual environment or [Anaconda](https://www.anaconda.com/) is recommended.
